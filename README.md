# Hoblio

Hoblio is a web-first modular tracker platform. The MVP is focused on the Cricket Tracker, with future predefined trackers added as developer-owned modules.

This is not a user-created template builder. Trackers are product-defined apps with their own routes, database tables, business logic, validation, dashboards, and future AI flows.

## Current Trackers

- Cricket Tracker: first MVP tracker
- Sneaker Vault: planned later, coming soon only

## Local Setup

```bash
npm install
npm run dev
```

Create `.env.local` from `.env.example` when Supabase and OpenAI credentials are available.
